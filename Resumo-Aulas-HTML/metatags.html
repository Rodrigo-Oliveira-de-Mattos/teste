O que é uma meta tag?

Pertencente ao head, é usado para definições da página como formatações, descrições, compatibilidade, etc. Ela é usada com tags dentro dela para adicionar tais características

O que é uma tag?

São etiquetas de orientação que são responsáveis por criar elementos com função de organizar, identificar e limitar o conteúdo de uma pagina

Algumas tags que ficam no head:

  • tittle → É o texto que aparece aparece na aba do navegador.
  • meta name = “description” → É o texto que vai aparecer abaixo do title, informando o que contém na pesquisa. Geralmente, esse texto está incompleto e tem que acessar a página para lê-lo por completo e ver a pesquisa toda.

Atributos:

  • charset → Define o formato de identificação da página, ou seja, quais caracteres serão aceitos.
  • UTF-8 → Formato UniCode, ou seja, formato padrão da página , cujo aceitará qualquer caractere de qualquer idioma
  • def: Essa tag vai oferecer instruções sobre como controlar as dimensões e a escala da página html
  • width=devide-width → define a largura da página que vai se adequar com a largura do dispositivo que o usuário está acessando, dependendo da largura do dispositivo.
  • incial-scale → define o nível de zoom da página quando ela é carregada, melhorando a visualização para telas menores.
  • Descreve o que o site faz → (explicada em “algumas tags que cicam no head”)
  • content → É no content que vai o texto que auxilia no mecanismo de pesquisa.
