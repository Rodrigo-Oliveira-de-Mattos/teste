Títulos e parágrafos 🅰️

Os títulos no HTML são representados pela tag h e um numero. O “h” significa heading e o número significa a relevância dessa tag na pagina

DICA: Use apenas um "h1" por página os demais podem ser usado com mais frequência desde que façam sentido dentro da página

Os parágrafos vão ser representados por p (sem número). Quando colocado outro parágrafo separado, por padrão o navegador vai dar um espaçamento

Formatação de texto 📰

Está proibido o uso de formatação apenas para deixar o texto “bonitinho”, isso é trabalho para o CSS, HTML é linguagem de MARCAÇÃO para SEMANTICA as tags tem que ter um significado forte.
