O que é um Arquivo Index? 📂

  • É um arquivo que fica na raiz do projeto
  • Padrão para todos os servidores
ATENÇÃO:
Servidores sempre buscam o index primeiro, sem o index o servidor não vai abrir o projeto

O que é um Servidor? 👾

  • Servidores são lugares onde vc vai colocar seus projetos para que ele fique acessivel para todo mundo

Estrutura base HTML 🏗️

CUIDADO:
Sempre fechar uma tag
  • !DOCTYPE html ← Serve para o navegador identificar que esse é um arquivo HTML (apenas HTML5)
  • html ← Todas as marcações ficarão aqui dentro
  • head ← Estruturas que não aparecem pro usuário final (exceto title). Na head estarão todas as definições (metatags, link do css, etc…)
  • body← Parte visível para o usuário final
  • title ← Titulo da pagina
